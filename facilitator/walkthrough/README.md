# Workshop walkthrough SPA

This folder contains a local, dependency-free SPA for presenting or self-guiding through the numbered workshop sections.

Run it from the repository root:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173/facilitator/walkthrough/
```

The Markdown files remain the source of truth. The SPA adds navigation, visual transitions, facilitator/learner cues, copy-ready prompts, and local progress tracking.
