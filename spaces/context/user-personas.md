# User personas

## Senior engineer

- Wants less repetitive work
- Cares about correctness, reviewability, and architecture consistency
- Will adopt Copilot if it saves time without creating churn

## Product-minded developer

- Wants help turning rough ideas into scoped tasks, specs, and first implementations
- Values brainstorming, synthesis, and structured output

## New team member

- Needs repo understanding, examples, and safe starter tasks
- Benefits most from explain-first and test-first interactions
